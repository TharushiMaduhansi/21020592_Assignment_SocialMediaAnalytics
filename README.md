# Social Media Engagement Analysis

This repository contains a **Jupyter Notebook (`analysis.ipynb`)** for analyzing social media engagement across multiple platforms and post types. The analysis includes data cleaning, exploratory data analysis (EDA), statistical tests, and regression modeling to identify trends and the best performing contentâ€“platform combinations.

**Dataset used:**  
https://www.kaggle.com/datasets/cagrickr/social-media-post-engagement-dataset  

---

## Requirements

To run the analysis, you will need the following Python libraries:

- pandas  
- numpy  
- seaborn  
- matplotlib  
- statsmodels  
- scipy  
- scikit-learn  
- plotly  
- bokeh  

---

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/TharushiMaduhansi/21020592_Assignment_SocialMediaAnalytics.git
cd 21020592_Assignment_SocialMediaAnalytics
```

### 2. Install Dependencies

After ensuring Python is installed, run the following command:

```bash
pip install pandas numpy seaborn matplotlib statsmodels scipy scikit-learn plotly bokeh
```

### 3. Run the Analysis

- Launch Jupyter Notebook or VS Code
- Open analysis.ipynb
- Ensure social_media_engagement_dataset.csv is in the same directory
- Select Run All Cells to execute the analysis
